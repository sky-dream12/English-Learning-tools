# 英语学习平台

一款沉浸式英语学习在线教育平台，支持手机端像APP一样运行。

## 功能特性

### 核心功能
- ✅ 分级课程体系（小学六年级 - 大学四年级）
- ✅ 单词记忆（卡片翻转学习）
- ✅ 语法练习（选择题）
- ✅ 口语跟读（语音合成）
- ✅ 听力训练
- ✅ 学习进度追踪（数据可视化）
- ✅ 用户注册/登录系统
- ✅ 个性化学习路径
- ✅ 社区交流
- ✅ 成就激励系统

### PWA支持
- 📱 手机端可像APP一样运行
- 🏠 添加到主屏幕
- 📴 离线缓存支持
- 📱 全屏模式体验

## 使用方法

### 电脑端使用
直接用浏览器打开 `index.html` 文件即可。

### 手机端使用（像APP一样）

#### iPhone/iOS
1. 用Safari浏览器打开 `index.html`
2. 点击分享按钮（方形带箭头图标）
3. 向下滑动找到并点击"添加到主屏幕"
4. 点击"添加"
5. 现在可以在主屏幕找到并像APP一样打开使用

#### Android
1. 用Chrome浏览器打开 `index.html`
2. 点击菜单按钮（三个点）
3. 点击"安装应用"或"添加到主屏幕"
4. 按照提示操作

## 技术栈

- HTML5 + CSS3 + JavaScript (ES6+)
- Tailwind CSS (CDN)
- Chart.js (数据可视化)
- Web Speech API (语音合成)
- Service Worker + Cache API (PWA)
- LocalStorage (数据存储)

## 分级课程

- 小学六年级
- 初一至初三
- 高一至高三
- 大学一年级至四年级
